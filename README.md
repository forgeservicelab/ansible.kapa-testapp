kapa-testapp
=========

Playbook for installing Secure Server for X-Road (Kansallinen Palveluväylä).

Requirements
------------

- Ubuntu 14.04 or later on target machine.

Role Variables
--------------

- `secure_server_address` - Address of secure server to POST request to.
